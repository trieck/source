Measure And Note Placement Rules

 

  1. The vertical placement of notes on the measure is directly related to the duration of the previous note in the measure.  If there are no previous related notes, then the note is aligned to the leftmost positional value.
  2. The leftmost positional value is effected by three items that may appear in the measure; the key signature, the time signature, and the clef.
  3. An attempt to add a note to a measure triggers a check against the current time signature and the duration of the notes in the measure to determine if there is space available in the measure.
  4. An attempt to move a note between two existing notes in a measure will cause the note on the right to shift to the right.  It’s placement will be determined by the duration of the new note.

Staff And Measure Placement Rules

 

  1. A staff has a fixed number of measures per system.
  2. A measure’s width is initially determined by the following factors:
    • The number of measures contained in the parent staff
    • The existence of a clef, key signature and time signature bitmap
